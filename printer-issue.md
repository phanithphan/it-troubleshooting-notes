# Paper Jam:
- Cause: mis-align paper in trays, over filling the tray, sometime using the wrong paper size.
- Solution: Slowly pull paper out of the paper path (avoid ripping it off), then check the paper alignment and the size of the paper that being used.

# Lower Toner Ink:
- Cause: printer commonly give us sign that the printer run out of the ink (but it's actually still have 10-20% supply left)
- Solution: in this case, if the printed page still not fading we can keep printing it. just remove the cartridge and give it a little shake, then carefully insert it back into the tuner.

# "Offline Status" on printer:
- Casue: when we hit print, but accidently nothing happened. then we check the printer status and it show "Offline"
- Solution: check the printer connection whether it's on the same connection with computer or not. If it doesn't work can try restart the printer and computer, then reconnect to the same network again.
